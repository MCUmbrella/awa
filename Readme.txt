Usage: awa <Number o' "awa"s> [[--upper || -u] || [--aupper || -U]]
-u or --upper: All of "awa" is in upper case
-U or --aupper: The 'a' character is in upper case
-d or --debug: No awa. Used only for debugging of type converter. May be removed in the future.

Exit codes:
0: No problem awa
1: Number o' "awa"s not acceptable
2: Too few arguments
3: Don't use '-u' and '-U' in one command
